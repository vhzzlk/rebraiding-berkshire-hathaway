# ⚡ QUICKSTART - Berkshire Redesign

## 1️⃣ Instalação Rápida (3 passos)

```bash
# Passo 1: Abra o terminal na pasta do projeto
cd caminho/para/berkshire-redesign

# Passo 2: Instale as dependências
npm install

# Passo 3: Inicie o projeto
npm run dev
```

✅ Pronto! O site abrirá em http://localhost:5173

---

## 🎨 Visualizar Seções

Navegue entre as seções:
- **Navbar** - Menu responsivo no topo
- **Hero** - Seção principal com gráfico
- **About** - Sobre a empresa
- **Investments** - Portfolio com gráfico em pizza
- **Shareholders** - Recursos para acionistas
- **Reports** - Relatórios financeiros
- **Footer** - Rodapé com links

---

## 🏗️ Build para Produção

```bash
npm run build
```

Arquivos otimizados estarão em `/dist`

---

## 📱 Testar Responsividade

- Use DevTools do navegador (F12)
- Clique no ícone de dispositivo móvel
- Teste em diferentes tamanhos
